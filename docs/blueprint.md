# **App Name**: Inkwell Musings

## Core Features:

- Content Editor & Uploader: An intuitive admin interface for the author to create, edit, and upload blog posts (including rich text, images, and categories) without needing to code.
- Dynamic Post Display: Clean and responsive display of individual blog posts, featuring titles, content, publication dates, and author information.
- Categorization & Tagging System: Allow posts to be categorized with predefined tags like 'Projects', 'Thoughts', 'Articles', 'Stories' for easy organization and filtering.
- Featured Posts Section: A prominent section on the homepage to highlight a selection of the author's most significant or recent writings.
- About the Author Page: A dedicated static page to provide information about the author, their journey, and aspirations.
- Theme Toggle: Implement a user-friendly toggle to switch between a dark and a light visual theme for comfortable reading.
- AI-Powered Newsletter Notifications: Enable readers to subscribe to a newsletter. An AI tool will automatically generate and send concise email notifications to subscribers when a new blog post is published.

## Style Guidelines:

- Light/Dark Scheme: Designed with a primary dark scheme for an introspective, literary feel, with full support for a contrasting light theme. Primary color is a deep, refined blue, making elements stand out elegantly on dark backgrounds (HSL(220, 60%, 45%), RGB #2E66BF).
- Background Color (Dark Scheme): A very dark, subtle blue-grey to evoke a sense of quiet focus (HSL(220, 15%, 8%), RGB #131517).
- Accent Color: A bright, clear cyan provides a crisp highlight for interactive elements and calls to action, ensuring visibility against both dark and light themes (HSL(190, 70%, 65%), RGB #75D6E6).
- Headline font: 'Playfair' (serif) for an elegant, fashionable, and high-end feel, reflecting an author's distinct voice.
- Body font: 'PT Sans' (humanist sans-serif) for clean readability and approachability, suitable for longer forms of text such as blog articles and stories.
- Use subtle, line-based icons that align with themes of writing, reflection, and project work, maintaining a modern and unobtrusive aesthetic.
- Implement a clean, uncluttered blog layout featuring a main content area and a minimalist sidebar, optimizing for readability with ample white space and full responsiveness across devices.
- Incorporate subtle animation for theme switching, smooth scrolling, and gentle hover effects on interactive elements to enhance user engagement without distraction.
